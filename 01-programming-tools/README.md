# Stage 1 — Programming & Data Tooling

## What This Stage Covers

Before you can implement ML algorithms from scratch, you need fluent Python and confidence with the scientific computing stack. This stage builds the programming foundation for everything that follows.

| Topic | Tools |
|-------|-------|
| Python | Functions, OOP, exceptions, file I/O, regex, unit testing (pytest), type hints |
| Numerical computing | NumPy — vectors, broadcasting, vectorized operations |
| Data wrangling | Pandas — DataFrames, groupby, merge, reshaping |
| Visualization | Matplotlib + Seaborn — static plots, statistical charts |
| Development environment | Jupyter notebooks, virtual environments |

---

## Learning Goals

By the end of Stage 1, you should be able to:

- ✅ Write clean, idiomatic Python with proper error handling and testing
- ✅ Implement vector and matrix operations in NumPy without loops
- ✅ Load, clean, reshape, and aggregate tabular data with Pandas
- ✅ Create informative visualizations (scatter, histogram, heatmap, pair plot)
- ✅ Work comfortably in Jupyter notebooks
- ✅ Read and understand scikit-learn's API (will be used in Stage 2)

## Notes

- Run CS50P problem sets in your own local environment, not just in the browser. Setting up VS Code + venv is itself a learning exercise.
- Kaggle Learn's browser-based Jupyter interface means zero setup — ideal for rapid tooling exposure.
- The Python Data Science Handbook is best read as a reference alongside Kaggle Learn, not cover-to-cover.
