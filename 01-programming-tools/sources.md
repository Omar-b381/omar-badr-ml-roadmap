# Stage 1 — Sources

## 🏆 Primary

### [CS50P — Introduction to Programming with Python (Harvard)](https://cs50.harvard.edu/python/)

- **Type:** Video course + weekly problem sets + final project (self-paced)
- **Cost:** Free (all lectures + problem sets). Verified certificate via edX: ~$149 (optional)
- **Math depth:** N/A (programming stage)
- **Why this one:** CS50P goes far deeper than language syntax — you learn proper Python idioms, exception handling, regex, unit testing with pytest, and OOP that are all directly relevant to data science workflows. Problem sets are famously rigorous, forcing active coding rather than passive consumption. Unlike freeCodeCamp's Python course (~14 hrs video, general-purpose), CS50P's challenge level primes you perfectly for Stage 2. Unlike Kaggle Learn's Python micro-course (~5 hrs, very shallow), CS50P ensures you can write clean, debuggable Python code. David Malan is one of the most acclaimed CS educators alive.
- **Prerequisite:** None — no prior coding required
- **Est. time:** ~50–90 hours (10 weeks × 5–9 hrs/week including problem sets)
- **Mirror (edX):** https://www.edx.org/learn/python/harvard-university-cs50-s-introduction-to-programming-with-python

## 📌 Supplementary 1

### [Kaggle Learn — Python + Pandas + Data Visualization](https://www.kaggle.com/learn)

- **Type:** Interactive browser-based notebooks + auto-graded exercises
- **Cost:** Free (100% — no signup required to browse)
- **Math depth:** N/A (tool-focused)
- **Why this one:** Perfectly bridges the gap between CS50P's Python fundamentals and Stage 2's data science needs. Covers exactly the tooling required: Pandas DataFrames, NumPy arrays, Matplotlib/Seaborn plotting — all inside Jupyter-like notebooks with zero install. The "no setup" browser environment is ideal for beginners.
- **Prerequisite:** Basic Python (complete CS50P first or in parallel)
- **Est. time:** ~12–15 hours combined
- **Direct links:**
  - [Python micro-course](https://www.kaggle.com/learn/python) (~5 hrs)
  - [Pandas micro-course](https://www.kaggle.com/learn/pandas) (~4 hrs)
  - [Data Visualization micro-course](https://www.kaggle.com/learn/data-visualization) (~3 hrs)

## 📌 Supplementary 2

### [Python Data Science Handbook — Jake VanderPlas](https://jakevdp.github.io/PythonDataScienceHandbook/)

- **Type:** Free online textbook with runnable Jupyter notebooks
- **Cost:** Free (full book online). GitHub: https://github.com/jakevdp/PythonDataScienceHandbook
- **Math depth:** Low-medium — introduces NumPy, Pandas, Matplotlib, and sklearn at a conceptual level
- **Why this one:** The most comprehensive free reference for the scientific Python stack. Matplotlib coverage is more thorough than any other free resource. The scikit-learn chapter provides a great bridge into Stage 2. Consistently listed on HackerNews, Reddit, and academic curricula as a "must-have" desk reference. Best used alongside or after Kaggle Learn (it has no problem sets — self-discipline required).
- **Prerequisite:** Basic Python (use after CS50P + Kaggle Learn)
- **Est. time:** ~30–50 hours to read and work through all notebooks

## ❌ Rejected Candidates

| Resource | Reason Rejected |
|----------|----------------|
| **freeCodeCamp Python** | "Python for Everybody" (~14 hrs) is too shallow for data science prep. "Data Analysis with Python" cert is reasonable but patchwork compared to the CS50P + Kaggle + VanderPlas triad |
| **fast.ai pre-course** | fast.ai's official "pre-course" materials are a short Python refresher (~2–3 hrs), not a full Stage 1 curriculum. The full Practical Deep Learning course requires ~1 year of experience. Better used as a bridge to deep learning after Stage 2 |
| **CS50P Certificate** | The certificate is optional and costs $149 — all learning content is free without it |
